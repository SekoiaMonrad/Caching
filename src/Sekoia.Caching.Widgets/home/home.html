How to use this application template

This template is intended for creating wigdet micro services.

A menu is automatically constructed from all pages located in scaffolds, and the structure should be as follows:

  • /scaffolds
    • /menu1
      • /menuitem11
      • /menuitem12
    • /menu2
      • /menuitem21
      • /menuitem22

Meaning that a dropdown menu will be created for each folder under scaffolds and a menu item will be created for each folder within those folders. The viewmodel file is expected to be named after menuitem folder.

Widgets to be included in the nuget package should be located in the widgets folder which follow Durandal2s default widget structure.
